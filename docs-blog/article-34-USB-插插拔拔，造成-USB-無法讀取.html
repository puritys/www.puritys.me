USB 插插拔拔，造成 USB 無法讀取

Tags:
  • Windows
簡中版

平常使用 Windows XP 的時候，當把 USB 拔掉後再插上去，"檔案總管" 或是"我的電腦" 就再也不會出現 USB 磁碟區了，通常只要重開機就解決這種鳥事，有一天心血來潮，花了一點時間研究一下，解決了這個問題，因為我不想再一直重開機.....

第一招： 電腦管理 - 磁碟管理

  • 先查看一下 Windows 有沒有抓到磁碟，並試著打開它。
  • 進入「開始」->「控制台」->「系統管理工具」->「電腦管理」->[磁碟管理]





  • 在這可以找到有啟動的磁碟，點新出現的 USB 磁碟區，按右鍵->「開啟」

這個方式可以讀取到資料夾，運作上也算正常，不過當我想把檔案備份到 USB ，在複製檔案的時候，卻出現一些問題，沒辦法抓資料複製到 USB 裡，系統會一直提示檔案名稱無法使用，所以這一招只成功了一半。

第二招： 重啟 Explorer

上一招失敗之後，突然想到檔案總管是由 explorer.exe 這個程序管理的，那就試著重啟 explorer.exe 把

  • 進入 Windows 工作管理員( 按 Ctrl+Alt+Del )，關掉 explorer.exe 這個 Process。


  • 進入Windows工作管理員，選「檔案」->「新工作(執行..)」，啟動新的 Process。


  • 輸入「explorer」，完成建立新工作

查看一下「我的電腦」，太帥了，USB磁碟區，重新出現了，而且操作一切正常無誤。

如果以上二招都無法解決，那麼你可能要找另外一台電腦來測試一下你的 USB 是不是壞掉了

用指令來確認 Window 是否有成功辨識 USB

如果 Window 一直沒辦法正確的讀到 USB 這個硬體，那麼還可以使用下指令的方式來檢查，首先選 開始 -> 執行 -> 輸入 cmd -> 按 Enter。

接著你就會看到一個黑黑的控制台畫面，這時輸入指令 :

wmic logicaldisk get deviceid,description

再檢查一下訊息，是否有出現 USB 的資訊，如果有看到 USB的資訊，那就是 Windows 還可以讀到 USB 裝置，這樣子的話，就可以試試 USB 還原軟體。

我把指令寫在一個 Windows 執行檔， 你可以直接下載這個檔案，點擊執行檔兩下就可以看到終端機的測試結果。

  • https://drive.google.com/file/d/0B1_zQGB-Kb7mekxCMWlaVFlvOUE/view?usp=sharing&resourcekey=0-asGN_A857Q9KEqArmTFXAg
