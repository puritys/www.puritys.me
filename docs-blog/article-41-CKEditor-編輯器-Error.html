CKEditor 編輯器 Error

Tags:
簡中版

最近使用 CKEditor，Javascript時發生奇怪的錯誤

錯誤訊息: uncaught exception: [CKEDITOR.editor] The instance "content" already exists.

原因是因為，我移除textarea後，重新 Create CKEditor，結果因為 content 這個物件已經被宣告過了，所以彈出錯誤訊息。

處理方式也很簡單，create CKEditor之前，先執行delete：

Example 
  1. <script>
  2. delete CKEDITOR.instances["content"];
  3. </script>

PS: content 是textarea的屬性name喔! 請自行修改。
