解決 Ionic 無法更新 Android App

Tags:
簡中版

在使用 Ionic 開發 IOS 與 Android App 時，我們都會使用指令 ionic build , ionic run，ionic run 執行的時候，它會去先移除 Android App 上的 APP，然後再安裝新版的程式，但是不知道為什麼，ionic run 無法正確的移除 Android App，造成新修改的程式無法正確更新，目前還沒有找到正解，只好先簡單寫一個 script 來自動移除 Android App。

ionic.sh 
  1. adb root
  2.  
  3. adb shell pm list packages | grep ionic |grep -o '[a-z\.0-9:]*' |sed -e 's/package://' | xargs -I%s -t -n 1  adb shell pm uninstall %s
  4.  
  5. ionic run

sh ionic.sh 執行這個指令就能成功 update Android App。

這個指令只能在 Mac 上運作。
