解決讀卡機偵測不到

Tags:
  • Win7
  • Windows
簡中版

在 Windows 7 環境下，有時候不知道為什麼，讀卡機會沒有反應，不管是插入 SD 卡，或是晶片卡，電腦都沒有反應，尤其是前幾天剛好要使用晶片卡轉帳繳水費，而且新北市的水費又只能用轉帳，沒辦法用信用卡來繳，結果好死不死的晶片卡怎麼也讀不到，害我跑了一趟便利商店繳費。

後來我發現解決問題的方式，就是將電腦重開機，當然這只是解法之一，不過我的電腦平常都不會關機，通常我都會使用冬眠 (Hibernation) 功能來將電腦關起來，這個方式很容易造成下次啟動讀卡機功能會失效，而重新開機就能解決這個問題。

又過了幾天，我再度研究了讀卡機相關功能，我發現只要重啟 Smart Cart 這個功能，就能修復讀卡機讀不到的問題。

首先我們打開控制台，選擇管理員工具 (Administrator Tool)，再點擊「服務」 (Services)，接著你會看到下面這個畫面。

點選 Smart Card 後，你會看到下面這個畫面，先將這個功能停用，再啟用，讀卡機的功能就能恢復正常了。
