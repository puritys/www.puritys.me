JSON Encode : 去掉 JSON 格式的空行

Tags:
  • Tools
簡中版

JSON 在 Javascript 就是一個 array ，有時候我會需要 stringify 的功能，也就是將 JSON 轉成文字，並且去掉所有的空行，

這是一個小小的工具，貼上你的 JSON 內容後，再點擊確認鈕，程式就會自動幫你把 JSON 轉成一行字串。

請輸入 JSON
"{ "food": [ "meat", "pork", "seafood" ] } "
