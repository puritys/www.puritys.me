如何 開啟 / 關閉 Firefox Browser Cache (快取)

Tags:
簡中版

快速關掉 Firefox Cache 的方式，請先打開 Firefox Browser ，然後在網址列輸入 "about:config" ， 輸入 Enter 後 ， Firefox 就會打開所有的設定值，最後點擊 "browser.cache.disk.enable" 就可以開關 Cache (快取) 設定。


官方資訊:

  • https://support.mozilla.org/en-US/kb/how-clear-firefox-cache
