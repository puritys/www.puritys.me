下載舊版 Google Chrome (保證無毒)

Tags:
  • Chrome
簡中版

網路上有很多舊版 Google Chrome 載點，但是你相信那些載點沒有被藏病毒或是木馬嗎?

commondatastorage.googleapis.com 這是 Google 官方提供的 Chrome 下載空間， 每個版本都有，而且都是免安裝版本，只要下載回來解壓縮就能使用，從這裡下載鐵定不會有病毒，如果你發現從這裡下載到有病毒的軟體，說不定還能跟 Google 要點賞錢。

以下是 Google Chrome 的下載網址：

  • 全部： https://commondatastorage.googleapis.com/chromium-browser-snapshots/index.html
  • Windows: https://commondatastorage.googleapis.com/chromium-browser-snapshots/index.html?prefix=Win/
  • Linux: https://commondatastorage.googleapis.com/chromium-browser-snapshots/index.html?prefix=Linux/
  • Mac: https://commondatastorage.googleapis.com/chromium-browser-snapshots/index.html?prefix=Mac/

如果版本太多你找不到的話，這裡提供我找到的幾個版本：

  • Windows chrome 68.0
    • https://commondatastorage.googleapis.com/chromium-browser-snapshots/index.html?prefix=Win_x64/552589/
  • Mac chrome 68.0
    • https://commondatastorage.googleapis.com/chromium-browser-snapshots/index.html?prefix=Mac/552589/

看到 chrome-win32-syms.zip ，檔名後面有帶 syms 代表開發版本，你不是開發人員就不要下載，通常我們只要下載 chrome-win32.zip 這種檔名就好。

備份

為了避免哪一天， Google 突然不提供下載，或是 Google 收掉了(應該是不太可能)，我自已備份了幾個版本：

  • Windows Chrome 41.0 ， Win XP 可以使用
    • https://drive.google.com/file/d/19XWT1VStt4ct-_CWmsSDQBg39uSn_Pdg/view?usp=sharing
  • Mac Chrome 68.0
    • https://drive.google.com/file/d/17ScuyumyAOQ_4MVgwl5d92BU3AiMozM8/view?usp=sharing
  • Windows 68.0
    • https://drive.google.com/file/d/1ZAXvNOgkiSPbOVPBHRsXCnEuRf5Jnzc6/view?usp=sharing
