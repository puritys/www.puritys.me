Mac 打開 terminal 超級慢

Tags:
  • Apple
簡中版

使用 Mac Retina 已經有一段時間了，這一陣子 Mac 的 terminal 非常的奇怪，剛打開一個新的 terminal 時，會先卡住一陣子，要等大約 2 ~ 3 分鐘才會正常，有時重新開機可以解決問題，但有時又不行。

後來在 Google 找了一下原因，大部分都是說要砍掉 Mac 的系統記錄。

砍掉 system log 的指令如下

sudo rm -rf /private/var/log/asl/*.asl
