HTML 符號表 NCR

Tags:
  • html
  • Tools
簡中版
有時候在使用HTML 的符號時，會因為編碼的關系，而有錯誤的顯示方式，這時我們可以使用HTML代碼，減少代碼相衝。
使用方式：先輸入「&#」，再接「數字;」，下列是一些的特殊符號區塊分類。

  • [55250為最後一號]   [12900 =㊖]   [9500~9700 ▲]
  • [10000~11800,5000~7800空白]
  • [9424=ⓐ]  
  • [9829~♥♦♧♨♩♪♫♬♭]
  • [12354=あ日文]  [12550=ㄆ注音]  [13184=㎀單位]
  • [19968=一中文，後面的都是中文]
  • 12958 ㊞ 12963 ㊣ ㋀ ㋁㋂㋃㋄

    ~
