我討厭 YUI 3 .....第十三天 Radio

Tags:
  • YUI3
簡中版

Html 中的 Form 表單，會很普遍的使用 Radio , Checkbox，但是當 User 選擇其中幾個選項後，我該如何用 Javascript 去取得該數值呢?

Radio 單選資料處理

假設我有一組 Radio Button ，我選了「台灣」這個選項，範例如下：



點我取得 Radio 的值

這時我們可以透過 :checked 取得 User 選擇的選項

Example 
  1. <script>
  2. YUI().use("node", function (Y) {
  3.     var value = Y.one('input[name=t]:checked').get('value');
  4.     Y.log(value);
  5. });
  6.  
  7. </script>

Checkbox 複選資料處理



點我取得 Checkbox 的值

Checkbox 我們一樣是透過 :checked 取得 User 選擇的選項，唯一不同的時，必須使用 Y.all()

Example 
  1. <script>
  2. YUI().use("node", function (Y) {
  3.     var value = Y.all('input[name=checkboxSameple]:checked').get('value');
  4.     Y.log(value);
  5. });
  6.  
  7. </script>
