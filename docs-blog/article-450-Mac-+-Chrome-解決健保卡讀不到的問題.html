Mac + Chrome 解決健保卡讀不到的問題

Tags:
簡中版
- https://eservice.nhi.gov.tw/Personal1/System/Documents/new/readme3.htm

上面這個連結是健保局提供的 Mac 安裝健保卡程式的網頁，按照步驟安裝完成後打開瀏覽器，並輸入 file:///usr/local/share/NHIICC/html/ESample.html 可以進入健保卡測試頁，點擊 Get Basic 就會看到這串文字 GetBasic:000:000;00:，內容還包含你的姓名，有看到姓名的話代表成功讀到你的健保卡。

如果很不幸的你看到 Response 欄位出現 "8013"，這代表軟體讀不到你的卡片，解決方式 重新插一次卡片，或是重新插讀卡機 ，再不行就重新安裝一次健保卡軟體 mac.Install1，還是讀不到的話就多安裝幾次健保卡軟體，真的不行就只能重新開機了。

重覆一次解決步驟，每試一個步驟都要記得測一次讀卡頁面

  • 重新插健保卡
  • 重新插讀卡機
  • 重新安裝健保卡軟體
  • 再一次重新安裝健保卡軟體
  • 減少轉接，讀卡機直接插 Mac ，不要接在 hub 上
  • 把不必要的 USB 裝置先拔掉
  • 重新開機

手動啟動健保卡讀取程式

如果你熟悉 Mac Terminal ，可以手動啟動健保卡程式，還能看到隱藏的 Log 喔 !!

一般在安裝完健保局的軟體後，Mac 系統會出現一個程式名為 macHC，使用 Terminal 輸入 ps aux |grep macHC 就可以看到。

Example 
  1. ps aux |grep macHC
  2. root             61306   0.0  0.0  4327580   2480 s006  S+   12:59PM   0:00.02 /usr/local/share/NHIICC/macHC

再確定硬體方面都沒問題後，有可能就是這個程式出了問題，試著重新啟動它，先用 kill command 關閉 macHC ，等幾十秒後，再輸入 sudo /usr/local/share/NHIICC/macHC 啟動健保卡程式，再重新插一次卡片測試。
