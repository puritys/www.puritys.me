巴塞隆納自助旅行第四天 - 凱旋門與城堡公園

Tags:
  • Barcelona
簡中版
第四天，早上就下大雨，狂打雷閃電，今天的天氣很特別，第一次在巴塞隆納遇到下雨，而且雨下得超大附加打雷，不過大約只下了三個小時，時間不長，又剛好集中在一大早睡覺時間，所以對旅遊行程影響不大，可是 WIFI 卻因為打雷壞掉了，這天早上老婆安排好先用洗衣機洗好衣服再出門，等待洗衣服的過程，除了吃吃早餐以外，還可以用 WIFI 查查後面幾天的遊玩景點，而 WIFI 壞掉對我們影響可是超大的，只能看著講阿拉伯/西班牙語的電視 XD，我們旅館的櫃台不是 24 小時有人在，只好打電話給總辦公室的服務人員，請他們盡快來處理，但是打完電話後，又等了一個小時， WIFI 仍然沒有修好，身為一個專業的軟體工程師，網路壞掉當然要自已修囉 !!

如何在歐洲旅館，修理 WIFI 網路

我的手機可以正確的連到 WIFI 路由器，但是卻不能上網，可見是路由器對外的連線出錯，也有可能是路由器上一層的主機有問題，一般路由器都會有個網頁版的管理介面，只要知道路由器的 IP ，就能連到那個管理介面，我就試著輸入 192.168.0.1 , 192.168.1.1 ，果然其中一個就是網頁路由器管理介面，管理介面一定要輸入管理員的密碼，我就隨手輸入 WIFI 的連線帳號，很好運，一次就猜中了，這網管人員應該要打屁股，Admin 密碼跟 WIFI 連線密碼竟然是一樣的，真是秒破解啊。



這台路由器 (Router) 是 Cisco 出產的 Linksys x3000 ，順利登入管理介面後，先簡單的檢查一下它的設定，它是用 DHCP 方式發 IP 給他底下的裝置，也就是我的手機，這一部分的功能是正常的，我的手機有正確的拿到 IP ， WIFI 訊號也正常，但就是無法連網。


這台路由器有 Traceroute 的功能，用來檢查路由器對外的網路，我試著用這個功能 trace 從路由器到 Google.com 的網路封包，很明顯的它卡住了，看來是這台路由器無法連到外面的網路。


再來檢查它有沒有設定 pppoe ，pppoe 就是我們平常輸入 中華電信的帳號/密碼，用來連線到中華電信主機的通訊方式，這台機器並沒有直接播號到 ISP (中華電信也是 ISP)，它是先連線到旅館裡上一層的主機，再由那台主機負責連接到外面的網路，我查到房間路由器對外的 IP Address，竟然是顯示 0.0.0.0 ，看來是上一層主機沒有正確的分配 IP 給這台路由器，另外我注意到管理介面右下角有一個 "Renew IP Address" 的按鈕，試著點點看，結果路由器的 IP 回來了 !!! 顯示 "192.168.1.33"，些時我的手機網路也好了，沒想到出國旅遊還要自已修理 WIFI ... 而且還修好了，真是有趣。

分析一下旅館的網路設定，最外面有一台主機，他的內部 IP 是 192.168.1.1 ，這台主機負責發送內部 IP 給底下的路由器，所以我房間內的路由器拿到 IP 192.168.1.33 ，再下一層是路由器本身有個內部 IP: 192.168.0.1 ，它負責發送 IP 給底下的手機裝置，所以我的手機會拿到一個內部 IP: 192.168.0.x ，問題發生在最外層的主機沒有發送 IP 給房間內的路由器，原因可能是打雷導致主機封包瞬斷。

修完網路後，又過了大約 40 個分鐘，辦公室那邊終於派人過來了，這位不知名員工說，因為打雷的關系，導致屋頂的主機故障，而他剛剛已經去過屋頂重開主機，現在的 WIFI 網路已經正常了，雖然我早就修好我們房間內的 WIFI 了，不過還是要跟他說聲謝謝，。

城堡公園、凱旋門

凱旋門在巴塞隆納市區的東方，離市區不遠，走路過去只要 20 分鐘，巴塞隆納的天空很藍，雲朵很低很漂亮，就算天氣不好，風景還是很美，當然要慢慢的走過去，看看那些漂亮的房子跟天空，這裡是一個很大的廣場，不是很熱門的景點，連小攤販也知道不用來這裡賣商品，人不是很多，大部分都是遊客，大家都對著這個門一直拍，凱旋門附近的蒼蠅有點多，尤其是越靠近城堡公園，蒼蠅就越多，令人很不舒服，那天有下雨，天氣也不算熱，不知道為什麼蒼蠅滿天飛，因為太不舒服了，地上也因為下雨而濕濕的，我們沒有停留太久就往城堡公園去。


一到城堡公園就看到兩個警察在檢查一個黑人，這裡的黑人蠻多的，數量上比警察多好幾倍，感覺不是一個很安全的地方，主要是那些黑人看起來不像遊客，我們有盡量遠離他們，公園深處有個噴水池，上面有好幾匹金色的馬雕像，看起來很華麗，

如果天氣好的話，應該蠻適合散步的，可惜這裡都是沙地，沙子一碰到下雨，就變成泥土了，所以下雨天千萬不要到城堡公園玩，而且這裡的蒼蠅都直接飛撲到你臉上去，我們只能趕快拍一拍照就閃人了。

La Chinata

回到舊城區去，這裡環境比較乾淨，也比城堡公園安全的多，我們到一間橄欖油專賣店叫 "La Chinata"，裡面有各式各樣的橄欖油，店員倒了六種不同口味的橄欖油給我們配麵包試吃，從味道比較淡的試到重口味，其中還有水果口味的橄欖油，我個人不喜歡重口味的橄欖油，會有點噁心，必竟它還是油啊，水果口味算是大眾比較能接受的，老婆跟我兩個人都選擇水果口味的橄欖油，另外店裡還有賣橄欖油做成的醬料，如青醬，在店的更深處還有賣很多橄欖油相關的肥皂與護膚用品，老婆在這家店逛了超過一個小時，在這一個小時當中，男店員除了一開始準備給我們試吃之外，他本人一直在講電話，沒有停過，西班牙人好像特別愛聊天，都不會停的，只要你不需要他的介紹，他就站在櫃台一直不斷的講電話，超級能聊的。

戰利品


Casa Gispert 吉斯裴特之家

接著下一站就是有 200 年歷史的老雜貨店，不過我們到的時候，看這個店並沒有開門，從外面往店裡看，裡面堆滿雜物，看起來是已經不再營業了，可是 Casa Gispert 旁邊有一家店，裝潢跟他一模一樣，而且兩家店是連在一起的，在外面觀察了一陣子，才發現這兩家店內部是通的，根本就是同一家店，似乎只是換了個店名，但是舊招牌依然留著，怕老客戶不認得?

進去前，我發現店外有個用日本寫的小紙條，看來這家店有蠻多日本觀光客會來的，進去後，老板都不理我們，可能他以為我們不會講英文吧，而我以為他只會講西班牙文，所以也不理他 XD ，不過其實裡面的店員還不錯，這家店賣的家西很多，有巧克力、餅乾、乾貨、辣椒，橄欖油，果醬，大部分的東西都是我看不太懂的，這裡比較不太適合我這種年輕人來，好傳統的地方 !!

晚餐 - 七道門

七道門，英文名是 7 portes ，來之前就聽說這家餐廳的服務態度不太好，要不是老婆堅持吃一次看看，我早就放棄它了，因為沒有訂位的關系，我們五點半左右就到了這家餐廳，裡面的座位很多，不怕沒位置，我們點了柳丁汁，沙拉、安康魚湯，海鮮燉飯跟一份波菜，這份波菜是我們在巴塞隆納唯一一次吃到的純青菜料理，而點柳丁汁單純是因為不想點酒，而水又比酒貴，只好點杯果汁來喝喝。

不得不說他們的麵包真好吃，點完餐後，他會先來一份麵包+橄欖油，他們的橄欖油有一種水果的香味，而安康魚湯這道料理也含有一份麵包+黃芥茉醬，在台灣我是不吃黃芥茉的，但是七道門的黃芥茉就只有香而已，再加上一點點的甜味，沒有嗆味，配上麵包是真的好吃 !!

服務態度是真的普通，但是我不覺不好，服務員只是很少說"謝謝"，以及不會主動的來服務你，但是我本來就不喜觀服務生一直跑過來問我這個那個的，所以剛好是我可以接受的點，吃飽結完帳出去時，會有一個彪形大漢幫我們開門，而這位人兄也不說一聲 "Thank You"，真的除了幫開門，完全不理你，到是我跟他說了一聲 Thank you ，這可能就是為什麼網路上會流傳這家店服務態度不好的原因吧。

超市

Finish 晚餐，回家的路上看到一家很大的超市，裡面有各式各樣的麵包跟奶製品，除了牛奶之外還有羊奶，而且他們的奶製品有分兩種，一種是放冰箱的只能保存一週左右，另一種是放在一般的架位上，可以保存 1 ~ 3 個月，老婆一直質疑為什麼牛奶可以放這麼久? 不過我是享受當下的人，管他三七二十一，買回家試喝就對了，我們兩種都買了，我個人覺得放在一般架位上的牛奶比較好喝，而羊奶的奶味會比牛奶重一些。


這家店是不能拍照的，不過我都拍了好幾張，才有一個店員用西班牙文跟我說不能拍照，不要誤會，我一句西班牙文都聽不懂，後來店員用手比了一下旁邊的禁止照像標誌我才懂的。
