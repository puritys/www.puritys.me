Linux Freebsd 登入問題

Tags:
簡中版

登入 Freebsd 系統若發生錯誤：/boot/defaults/loader.conf unable to load kernel OK

代表系統抓不到kernel

解決辦法是

  • 方法一

自已指定kernel名稱，先進入freebsd 再修改，先下指令 boot kernel名稱可以執行舊的kernel，如：

  1. OK boot /boot/xxxkernel

不知道你 有什麼kernel的話，輸入 OK ls /boot 會自動列出一堆東西，那一個是正確的 kernel 呢? 自已猜吧 ... (一個一個試也行) ，我使用freebsd7.0 下這句有效

  1. OK boot /boot/GENERIC
  • 方法二

裝 freebsd live CD 光碟 登入去修改，當你順利登入freebsd後，修改開機檔 vi /boot/defaults/loader.conf ， bootfile="指定你要的kernel" #kernel name
