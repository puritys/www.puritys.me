Quotes

有些重要文字，我們會用雙引號將文字括起來，通常用到 ＜q> 這個 tag ，而我們可以透過 quotes 將 「"」 自動改成其他符號。

Quotes Sample 
  1. <style>
  2.    .q {
  3.         quotes: '«' '»';
  4.     }
  5.     .q:before {
  6.         content: open-quote;
  7.     }
  8.     .q:after {
  9.         content: close-quote;
  10.     }
  11. </style>
  12. <q>範例</q>
