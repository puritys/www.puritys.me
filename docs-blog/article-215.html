PHP Array 在經過 json_encode 之後，就會把中文轉成 Unicode 的代碼，看到那些代碼卻不知道代表什麼意思，這個問題，對於中文網站的開發者來說，是非常的困擾，所以我寫了一個簡單的工具，可以將 Unicode 轉成正常的 ASCII 中文文字。

PHP Unicode to Ascii

Example 
  1. $str = "\u6210\u529f";
  2. $str = preg_replace('/\\\u([0-9a-f]+)/i', '&#x$1;', $str);
  3. echo html_entity_decode($str, 0, 'UTF-8');
