FreeBsd 安裝與設定

Tags:
  • Linux
簡中版

網路設定

OS 裝好後的第一步，就是設定網路，首先我有一個網卡 em0 要連上網路，另外有一個網卡 em1 當作內部 IP 使用。

FreeBsd 的網路設定資料都存在 /etc/rc.conf，以下是基本的網路重啟指令。

  • 重啟全部網路指令 service netif restart
  • 路由重啟 service routing restart
  • 關閉單一網卡指令 ifconfig em1 down
  • 開啟單一網卡指令 ifconfig em1 up

DNS 設定

DNS 資料是存在 /etc/resolv.conf

resolv.con 範例 
  1. nameserver 168.192.95.1
  • 檢查網路狀態 netstat -rn

靜態 IP 設定

vi /etc/rc.conf，設定好重啟 OS

/etc/rc.conf 範例 
  1. ifconfig_em0="DHCP"
  2. ifconfig_em1="inet 192.168.56.12 netmask 255.255.255.0"
  3. gateway_enable="YES"
  4. static_routes="net2"
  5. route_net2="-net 192.168.56.0/24 192.168.56.1"

基本軟體安裝

安裝 sudo ，為了讓非 root 的帳號可以使用 root 的權限。

  • cd /usr/ports/security/sudo/ && make install clean
  • visudo : 使用這個指令，可以修改 sudo user ， 將你的帳號輸入 「account ALL=(ALL) ALL」 即可擁有 root 權限

安裝 rpm，方便安裝 rpm 的軟體

  • cd /usr/ports/archivers/rpm && make install clean

安裝 pkg， 這是 FreeBSD 的軟體安裝程式。

  • cd /usr/ports/ports-mgmt/pkg && make install clean

安裝文字編輯器

  • cd /usr/ports/editors/nano && make install clean
  • cd /usr/ports/editors/vim && make install clean

其他相關軟體

  • /usr/ports/distfiles/perl && make install clean

VirtualBox

VirtualBox NAT 網卡的資料如下

  • 預設 NAT ip = 10.0.2.2
  • NAT DHCP Server ip 10.0.2.0
  • broadcast 10.0.2.255
