TeaJS 網頁開發程式設計


在 2012年07月18日 、 v8cgi 改名為 TeaJS，不太清楚為什麼原作者要改名子，v8cgi 語意上是比較好懂的，就很像是 php cgi , perl cgi 這些東西。

現在改成 TeaJS ，好像有點抄 coffee script js ，或是抄 java 的咖啡呢，不管怎麼樣，東西好用就行了，名稱也不是這麼重要。

TeaJS 是一個 server 端使用的 Javascript ，可以單單使用 Javascript 來製作網站、App、伺服器等，底層使用了 Google 開發的 v8 engine ，可以搭配 apache 網頁伺服器一起使用，前後端同時用 Javascript 來開發，拉近前端工程師與後端工程師的界線，真是一個好物!!!
