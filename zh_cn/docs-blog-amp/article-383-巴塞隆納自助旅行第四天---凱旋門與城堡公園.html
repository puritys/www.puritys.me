巴塞隆纳自助旅行第四天 - 凯旋门与城堡公园

Tags:
  • Barcelona
繁中版

网页好读版

第四天，早上就下大雨，狂打雷闪电，今天的天气很特别，第一次在巴塞隆纳遇到下雨，而且雨下得超大附加打雷，不过大约只下了三个小时，时间不长，又刚好集中在一大早睡觉时间，所以对旅游行程影响不大，可是 WIFI 却因为打雷坏掉了，这天早上老婆安排好先用洗衣机洗好衣服再出门，等待洗衣服的过程，除了吃吃早餐以外，还可以用 WIFI 查查后面几天的游玩景点，而 WIFI 坏掉对我们影响可是超大的，只能看著讲阿拉伯/西班牙语的电视 XD，我们旅馆的柜台不是 24 小时有人在，只好打电话给总办公室的服务人员，请他们尽快来处理，但是打完电话后，又等了一个小时， WIFI 仍然没有修好，身为一个专业的软体工程师，网路坏掉当然要自已修罗 !!

如何在欧洲旅馆，修理 WIFI 网路

我的手机可以正确的连到 WIFI 路由器，但是却不能上网，可见是路由器对外的连线出错，也有可能是路由器上一层的主机有问题，一般路由器都会有个网页版的管理介面，只要知道路由器的 IP ，就能连到那个管理介面，我就试著输入 192.168.0.1 , 192.168.1.1 ，果然其中一个就是网页路由器管理介面，管理介面一定要输入管理员的密码，我就随手输入 WIFI 的连线帐号，很好运，一次就猜中了，这网管人员应该要打屁股，Admin 密码跟 WIFI 连线密码竟然是一样的，真是秒破解啊。



这台路由器 (Router) 是 Cisco 出产的 Linksys x3000 ，顺利登入管理介面后，先简单的检查一下它的设定，它是用 DHCP 方式发 IP 给他底下的装置，也就是我的手机，这一部分的功能是正常的，我的手机有正确的拿到 IP ， WIFI 讯号也正常，但就是无法连网。


这台路由器有 Traceroute 的功能，用来检查路由器对外的网路，我试著用这个功能 trace 从路由器到 Google.com 的网路封包，很明显的它卡住了，看来是这台路由器无法连到外面的网路。


再来检查它有没有设定 pppoe ，pppoe 就是我们平常输入 中华电信的帐号/密码，用来连线到中华电信主机的通讯方式，这台机器并没有直接播号到 ISP (中华电信也是 ISP)，它是先连线到旅馆里上一层的主机，再由那台主机负责连接到外面的网路，我查到房间路由器对外的 IP Address，竟然是显示 0.0.0.0 ，看来是上一层主机没有正确的分配 IP 给这台路由器，另外我注意到管理介面右下角有一个 "Renew IP Address" 的按钮，试著点点看，结果路由器的 IP 回来了 !!! 显示 "192.168.1.33"，些时我的手机网路也好了，没想到出国旅游还要自已修理 WIFI ... 而且还修好了，真是有趣。

分析一下旅馆的网路设定，最外面有一台主机，他的内部 IP 是 192.168.1.1 ，这台主机负责发送内部 IP 给底下的路由器，所以我房间内的路由器拿到 IP 192.168.1.33 ，再下一层是路由器本身有个内部 IP: 192.168.0.1 ，它负责发送 IP 给底下的手机装置，所以我的手机会拿到一个内部 IP: 192.168.0.x ，问题发生在最外层的主机没有发送 IP 给房间内的路由器，原因可能是打雷导致主机封包瞬断。

修完网路后，又过了大约 40 个分钟，办公室那边终於派人过来了，这位不知名员工说，因为打雷的关系，导致屋顶的主机故障，而他刚刚已经去过屋顶重开主机，现在的 WIFI 网路已经正常了，虽然我早就修好我们房间内的 WIFI 了，不过还是要跟他说声谢谢，。

城堡公园、凯旋门

凯旋门在巴塞隆纳市区的东方，离市区不远，走路过去只要 20 分钟，巴塞隆纳的天空很蓝，云朵很低很漂亮，就算天气不好，风景还是很美，当然要慢慢的走过去，看看那些漂亮的房子跟天空，这里是一个很大的广场，不是很热门的景点，连小摊贩也知道不用来这里卖商品，人不是很多，大部分都是游客，大家都对著这个门一直拍，凯旋门附近的苍蝇有点多，尤其是越靠近城堡公园，苍蝇就越多，令人很不舒服，那天有下雨，天气也不算热，不知道为什么苍蝇满天飞，因为太不舒服了，地上也因为下雨而湿湿的，我们没有停留太久就往城堡公园去。


一到城堡公园就看到两个警察在检查一个黑人，这里的黑人蛮多的，数量上比警察多好几倍，感觉不是一个很安全的地方，主要是那些黑人看起来不像游客，我们有尽量远离他们，公园深处有个喷水池，上面有好几匹金色的马雕像，看起来很华丽，

如果天气好的话，应该蛮适合散步的，可惜这里都是沙地，沙子一碰到下雨，就变成泥土了，所以下雨天千万不要到城堡公园玩，而且这里的苍蝇都直接飞扑到你脸上去，我们只能赶快拍一拍照就闪人了。

La Chinata

回到旧城区去，这里环境比较干净，也比城堡公园安全的多，我们到一间橄榄油专卖店叫 "La Chinata"，里面有各式各样的橄榄油，店员倒了六种不同口味的橄榄油给我们配面包试吃，从味道比较淡的试到重口味，其中还有水果口味的橄榄油，我个人不喜欢重口味的橄榄油，会有点恶心，必竟它还是油啊，水果口味算是大众比较能接受的，老婆跟我两个人都选择水果口味的橄榄油，另外店里还有卖橄榄油做成的酱料，如青酱，在店的更深处还有卖很多橄榄油相关的肥皂与护肤用品，老婆在这家店逛了超过一个小时，在这一个小时当中，男店员除了一开始准备给我们试吃之外，他本人一直在讲电话，没有停过，西班牙人好像特别爱聊天，都不会停的，只要你不需要他的介绍，他就站在柜台一直不断的讲电话，超级能聊的。

战利品


Casa Gispert 吉斯裴特之家

接著下一站就是有 200 年历史的老杂货店，不过我们到的时候，看这个店并没有开门，从外面往店里看，里面堆满杂物，看起来是已经不再营业了，可是 Casa Gispert 旁边有一家店，装潢跟他一模一样，而且两家店是连在一起的，在外面观察了一阵子，才发现这两家店内部是通的，根本就是同一家店，似乎只是换了个店名，但是旧招牌依然留著，怕老客户不认得?

进去前，我发现店外有个用日本写的小纸条，看来这家店有蛮多日本观光客会来的，进去后，老板都不理我们，可能他以为我们不会讲英文吧，而我以为他只会讲西班牙文，所以也不理他 XD ，不过其实里面的店员还不错，这家店卖的家西很多，有巧克力、饼干、干货、辣椒，橄榄油，果酱，大部分的东西都是我看不太懂的，这里比较不太适合我这种年轻人来，好传统的地方 !!

晚餐 - 七道门

七道门，英文名是 7 portes ，来之前就听说这家餐厅的服务态度不太好，要不是老婆坚持吃一次看看，我早就放弃它了，因为没有订位的关系，我们五点半左右就到了这家餐厅，里面的座位很多，不怕没位置，我们点了柳丁汁，沙拉、安康鱼汤，海鲜炖饭跟一份波菜，这份波菜是我们在巴塞隆纳唯一一次吃到的纯青菜料理，而点柳丁汁单纯是因为不想点酒，而水又比酒贵，只好点杯果汁来喝喝。

不得不说他们的面包真好吃，点完餐后，他会先来一份面包+橄榄油，他们的橄榄油有一种水果的香味，而安康鱼汤这道料理也含有一份面包+黄芥茉酱，在台湾我是不吃黄芥茉的，但是七道门的黄芥茉就只有香而已，再加上一点点的甜味，没有呛味，配上面包是真的好吃 !!

服务态度是真的普通，但是我不觉不好，服务员只是很少说"谢谢"，以及不会主动的来服务你，但是我本来就不喜观服务生一直跑过来问我这个那个的，所以刚好是我可以接受的点，吃饱结完帐出去时，会有一个彪形大汉帮我们开门，而这位人兄也不说一声 "Thank You"，真的除了帮开门，完全不理你，到是我跟他说了一声 Thank you ，这可能就是为什么网路上会流传这家店服务态度不好的原因吧。

超市

Finish 晚餐，回家的路上看到一家很大的超市，里面有各式各样的面包跟奶制品，除了牛奶之外还有羊奶，而且他们的奶制品有分两种，一种是放冰箱的只能保存一周左右，另一种是放在一般的架位上，可以保存 1 ~ 3 个月，老婆一直质疑为什么牛奶可以放这么久? 不过我是享受当下的人，管他三七二十一，买回家试喝就对了，我们两种都买了，我个人觉得放在一般架位上的牛奶比较好喝，而羊奶的奶味会比牛奶重一些。


这家店是不能拍照的，不过我都拍了好几张，才有一个店员用西班牙文跟我说不能拍照，不要误会，我一句西班牙文都听不懂，后来店员用手比了一下旁边的禁止照像标志我才懂的。
