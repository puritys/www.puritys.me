Mac + Chrome 解决健保卡读不到的问题

Tags:
繁中版

网页好读版

- https://eservice.nhi.gov.tw/Personal1/System/Documents/new/readme3.htm

上面这个连结是健保局提供的 Mac 安装健保卡程式的网页，按照步骤安装完成后打开浏览器，并输入 file:///usr/local/share/NHIICC/html/ESample.html 可以进入健保卡测试页，点击 Get Basic 就会看到这串文字 GetBasic:000:000;00:，内容还包含你的姓名，有看到姓名的话代表成功读到你的健保卡。

如果很不幸的你看到 Response 栏位出现 "8013"，这代表软体读不到你的卡片，解决方式 重新插一次卡片，或是重新插读卡机 ，再不行就重新安装一次健保卡软体 mac.Install1，还是读不到的话就多安装几次健保卡软体，真的不行就只能重新开机了。

重覆一次解决步骤，每试一个步骤都要记得测一次读卡页面

  • 重新插健保卡
  • 重新插读卡机
  • 重新安装健保卡软体
  • 再一次重新安装健保卡软体
  • 减少转接，读卡机直接插 Mac ，不要接在 hub 上
  • 把不必要的 USB 装置先拔掉
  • 重新开机

手动启动健保卡读取程式

如果你熟悉 Mac Terminal ，可以手动启动健保卡程式，还能看到隐藏的 Log 喔 !!

一般在安装完健保局的软体后，Mac 系统会出现一个程式名为 macHC，使用 Terminal 输入 ps aux |grep macHC 就可以看到。

Example 
  1. ps aux |grep macHC
  2. root             61306   0.0  0.0  4327580   2480 s006  S+   12:59PM   0:00.02 /usr/local/share/NHIICC/macHC

再确定硬体方面都没问题后，有可能就是这个程式出了问题，试著重新启动它，先用 kill command 关闭 macHC ，等几十秒后，再输入 sudo /usr/local/share/NHIICC/macHC 启动健保卡程式，再重新插一次卡片测试。
