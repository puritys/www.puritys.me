Java 安装方式与其变数型态

Tags:
繁中版

到 Java 官网下载页中，可以看到很多版本的 Java 软体，其中含有 SDK, JDK, JRE, J2EE、J2SE、J2ME 等等。

  • SDK: Sftware Develop Kit
  • JDK: Java Develop Kit
  • JRE: Java Runtime environment ，JRE 包含了 JVM: java virtual machine，只要我们有安装 JRE，就可以执行 Java 的程式，但是不包含编译功能。
  • J2EE: Java 2 企业版 (Enterprise Edition)，后来改名为 Java EE
  • J2SE: Java 2 标准版 (Standard Edition)，后来改名为 Java SE
  • J2ME: Java 2 简易版 (Micro Edition)，后来改名为 Java ME

一般工程师想要开发 Java 程式，只要下载 JDK 即可。

基本变数型态

type value max-min
boolean true,false 1 bit
char 'a' 16-bit = 2 byte

int -1,0,1 32-bit, -2^31 ~ 2^31 -1
long -1,0,1 64-bit, -2^63 ~ 2^63 -1
float 1.522
double 2.12345
