CKEditor 编辑器 Error

Tags:
繁中版

最近使用 CKEditor，Javascript时发生奇怪的错误

错误讯息: uncaught exception: [CKEDITOR.editor] The instance "content" already exists.

原因是因为，我移除textarea后，重新 Create CKEditor，结果因为 content 这个物件已经被宣告过了，所以弹出错误讯息。

处理方式也很简单，create CKEditor之前，先执行delete：

Example 
  1. <script>
  2. delete CKEDITOR.instances["content"];
  3. </script>

PS: content 是textarea的属性name喔! 请自行修改。
