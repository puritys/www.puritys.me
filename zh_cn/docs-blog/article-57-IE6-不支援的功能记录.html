Tags:

繁中版

css 变数名称底线开头

在IE6中，不能使用底线(_)开头的css名称

Example 
  1.   ._cssname{
  2.    color:red;
  3.  }
  4.  

下面这个范例，在 IE6 中看到的文字颜色会是错的。

Red text test.

css inline-block

IE6 不支援 display: inline-block 的语法，不过可以用 inline + zoom 解决这个问题。

Example 
  1. .inlineBlock {
  2.     display: inline-block;
  3.     zoom: 1;
  4.     *display: inline;
  5. }
