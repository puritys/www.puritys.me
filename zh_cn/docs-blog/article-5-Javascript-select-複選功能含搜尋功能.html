Javascript select 复选功能含搜寻功能

Tags:
繁中版

使用Select预设功能，来进行多重资料选取的工具，输出的画面放成三个部分，上半部为搜寻功能，可以从尚未选择的资料中进行搜寻，左半部为可以选的资料，右半部为已选择的资料。

选单范例

UI使用说明

Example 
  1.  
  2. <div id="text">  
  3. </div>  
  4. <script>  
  5. var option=new Array(  
  6. "1:0:日式牛排","2:0:酸酱牛排","3:0:牛肉面","4:1:酸辣面","5:0:杂酱面",  
  7. "6:0:清蒸鱼","7:0:煎鱼","8:1:白带鱼","9:0:牛小排","10:0:猪肉片",  
  8. "11:0:地瓜叶","12:1:高丽菜","13:0:花耶菜","14:0:A菜","15:0:大陆妹",  
  9. "16:0:西瓜","17:0:凤梨","18:1:莲雾","19:0:香蕉","20:0:综合拼盘"  
  10. );  
  11. var a=new multipleSelect({parent: "text" ,option:option,name:"test"},{showCanSelect:"可选择的菜单",showSelected:"已选择的菜单"});  
  12.   
  13. a.create();  
  14. </script>
  • 先将可以选择的资料放进Array，资料格式「1:0:name」：第一个资料是指id,第二个资料是指是否已经选择，1代表已选择，0代表尚未选择，第三个资料是物件名称。
  • 再建立multipleSelect物件，指定要插入的HTML TAG(id名称)，并可修改语言。
  • 最后执行建立，xxx.create()。
