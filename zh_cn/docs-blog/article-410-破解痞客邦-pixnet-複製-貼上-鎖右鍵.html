破解痞客邦 pixnet 复制/贴上/锁右键

Tags:
  • 破解
繁中版

有时候在网路上找一些餐厅吃饭，通常找到后会需要复制地址，贴到 Google Maps 上，但是有不少部落客会使用痞客邦 (常打成匹克帮)，然后又限制用户不能复制，不能点右键，连我只是要复制地址都不行。

破解方式很简单，就是执行以下四行程式码，如果你是使用 Chrome 浏览器，就打开"开发者工具" 或按 "Ctrl + Shift + i" (windows) ， "Cmd + Opt + i" (Mac) ，再把下列四行程式码贴到 console 上按 Enter 执行即可 。

Example 
  1. document.body.oncontextmenu="return true";
  2. document.body.onselectstart="return true";
  3. document.body.ondragstart="return true";
  4. document.body.style="";

tampermonkeyScript

每次打开痞客邦页面就要做一次这个动作也很麻烦，而且过两天就忘记程式码内容了，还要重新上网找一次，平常我都会安装 tampermonkey 这个外挂 ，这个外挂可以自动执行指定的程式码，把下面这个 github 的程式码贴上 tampermonkey userscript 就可以了，以后程式会自动帮你破解，你就不用再手动做了， Firefox 也有类似的外挂可以使用。

  • https://github.com/puritys/tampermonkeyScript/blob/master/src/pixnet.js
