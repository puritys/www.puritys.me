HTML 符号表 NCR

Tags:
  • html
  • Tools
繁中版
有时候在使用HTML 的符号时，会因为编码的关系，而有错误的显示方式，这时我们可以使用HTML代码，减少代码相冲。
使用方式：先输入「&#」，再接「数字;」，下列是一些的特殊符号区块分类。

  • [55250为最后一号]   [12900 =㊖]   [9500~9700 ▲]
  • [10000~11800,5000~7800空白]
  • [9424=ⓐ]  
  • [9829~♥♦♧♨♩♪♫♬♭]
  • [12354=あ日文]  [12550=ㄆ注音]  [13184=㎀单位]
  • [19968=一中文，后面的都是中文]
  • 12958 ㊞ 12963 ㊣ ㋀ ㋁㋂㋃㋄
