我讨厌 YUI 3 .....第十三天 Radio

Tags:
  • YUI3
繁中版

Html 中的 Form 表单，会很普遍的使用 Radio , Checkbox，但是当 User 选择其中几个选项后，我该如何用 Javascript 去取得该数值呢?

Radio 单选资料处理

假设我有一组 Radio Button ，我选了「台湾」这个选项，范例如下：



点我取得 Radio 的值

这时我们可以透过 :checked 取得 User 选择的选项

Example 
  1. <script>
  2. YUI().use("node", function (Y) {
  3.     var value = Y.one('input[name=t]:checked').get('value');
  4.     Y.log(value);
  5. });
  6.  
  7. </script>

Checkbox 复选资料处理



点我取得 Checkbox 的值

Checkbox 我们一样是透过 :checked 取得 User 选择的选项，唯一不同的时，必须使用 Y.all()

Example 
  1. <script>
  2. YUI().use("node", function (Y) {
  3.     var value = Y.all('input[name=checkboxSameple]:checked').get('value');
  4.     Y.log(value);
  5. });
  6.  
  7. </script>
