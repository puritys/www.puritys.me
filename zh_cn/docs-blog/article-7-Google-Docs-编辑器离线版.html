Google Docs 编辑器离线版

Tags:
繁中版

Docs的编辑器制作得跟Microsoft word很像，很适合给不懂HTML的部落客来进行网页编辑，这里仅提供Docs的离线测试版本给大家试玩。

这是笔者目前玩过，最好用的编辑器，比起 fckeditor ,ckeditor ,tinyMce ..等的编辑器，Google Docs 画面最干净，操作也是最直觉，大多的功能，都能在三个步骤内处理完成，另外最强的就是，它竟然有同Word的缩排功能，这对於网页排版非常的实用，当然也因为Docs做得太过精致，导致原始码过於冗长。
