Facebook FQL API 测试

Tags:
繁中版
Facebook 已经把 FQL 结束掉了，不能再使用了

FQL 是 Facebook 做的一种类 Sql 的语法，全名是 Facebook Query Language，我们可以用 Sql 语法取得 Facebook 各种资料，例如会员资料，Like 数， Share 数。

下面这个是 FQL 范例，这个范例可以取得我的 Blog 文章中的 like, share 数。

https://api.facebook.com/method/fql.query?format=json&query=select%20like_count,%20click_count,%20comment_count,comments_fbid,%20commentsbox_count,total_count,%20share_count%20from%20link_stat%20where%20url=%27http://www.puritys.me/docs-blog/article-151%27


测试工具，你可以用以下这个工具测试一下 FQL Api 。
