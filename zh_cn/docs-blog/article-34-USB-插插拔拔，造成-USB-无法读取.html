USB 插插拔拔，造成 USB 无法读取

Tags:
  • Windows
繁中版

平常使用 Windows XP 的时候，当把 USB 拔掉后再插上去，"档案总管" 或是"我的电脑" 就再也不会出现 USB 磁碟区了，通常只要重开机就解决这种鸟事，有一天心血来潮，花了一点时间研究一下，解决了这个问题，因为我不想再一直重开机.....

第一招： 电脑管理 - 磁碟管理

  • 先查看一下 Windows 有没有抓到磁碟，并试著打开它。
  • 进入「开始」->「控制台」->「系统管理工具」->「电脑管理」->[磁碟管理]





  • 在这可以找到有启动的磁碟，点新出现的 USB 磁碟区，按右键->「开启」

这个方式可以读取到资料夹，运作上也算正常，不过当我想把档案备份到 USB ，在复制档案的时候，却出现一些问题，没办法抓资料复制到 USB 里，系统会一直提示档案名称无法使用，所以这一招只成功了一半。

第二招： 重启 Explorer

上一招失败之后，突然想到档案总管是由 explorer.exe 这个程序管理的，那就试著重启 explorer.exe 把

  • 进入 Windows 工作管理员( 按 Ctrl+Alt+Del )，关掉 explorer.exe 这个 Process。


  • 进入Windows工作管理员，选「档案」->「新工作(执行..)」，启动新的 Process。


  • 输入「explorer」，完成建立新工作

查看一下「我的电脑」，太帅了，USB磁碟区，重新出现了，而且操作一切正常无误。

如果以上二招都无法解决，那么你可能要找另外一台电脑来测试一下你的 USB 是不是坏掉了

用指令来确认 Window 是否有成功辨识 USB

如果 Window 一直没办法正确的读到 USB 这个硬体，那么还可以使用下指令的方式来检查，首先选 开始 -> 执行 -> 输入 cmd -> 按 Enter。

接著你就会看到一个黑黑的控制台画面，这时输入指令 :

wmic logicaldisk get deviceid,description

再检查一下讯息，是否有出现 USB 的资讯，如果有看到 USB的资讯，那就是 Windows 还可以读到 USB 装置，这样子的话，就可以试试 USB 还原软体。

我把指令写在一个 Windows 执行档， 你可以直接下载这个档案，点击执行档两下就可以看到终端机的测试结果。

  • https://drive.google.com/file/d/0B1_zQGB-Kb7mekxCMWlaVFlvOUE/view?usp=sharing&resourcekey=0-asGN_A857Q9KEqArmTFXAg
