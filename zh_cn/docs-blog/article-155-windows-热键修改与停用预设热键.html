windows 热键修改与停用预设热键

Tags:
  • Win7
繁中版

windows 预设有不少好用的快捷键，如 win+L , win+D

Ctrl 右边那个按键，windows key 简称为 win，。

下面有几个软体，都是用来修改 windows 预设的热键

  • http://www.skynergy.com/hotkeyz.html
  • http://howto.cnet.com/8301-11310_39-20079677-285/how-to-disable-your-keyboard-windows-key/

这里有 Microsoft 对 window key 的代码说明。

  • http://msdn.microsoft.com/en-us/library/dd375731(VS.85).aspx

如何停用 windows 预设的热键

win+L 预设会执行登出的动作，有时候不小心按到，或是 win key 坏掉的时候，这个热键会非常的讨人厌，而且很多市面上免费的改热键程式，都无法停用这个热键，还好 Microsoft 有提供停用的服务，只要下载下列的程式，就可以停用 win key 的功能，也因为这是微软自已出的程式，所以不用害怕病毒，用力的装下去吧

Disable

  • http://go.microsoft.com/?linkid=9738719 (下载并安装)

Enable

  • http://go.microsoft.com/?linkid=9738718 (下载并安装)

使用　AutoHotkey 来停用 win key

  • 先下载 AutoHotkey 并安装　https://autohotkey.com/

在 AutoHotkey 加入下面两个设定，将左边的 win key 与右边的 win key 停用。

Example 
  1. Lwin::return
  2. Rwin::return
