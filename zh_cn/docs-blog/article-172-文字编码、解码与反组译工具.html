文字编码、解码与反组译工具

Tags:
繁中版

网路上常见的编码技术有 urlEncode, base64Encode, base32, md5, sha1, 等等，这些技术大部分的程式语言都内建已经支援了，例如 PHP 语言可以使用 base64_encode() , urlencode()，来对文字进行编码。

下面提供一些线上工具，方便快速执行解码的动作，这样不用每次想要解码时，就得打开 IDE 写一小段程式。

加解密工具

Online urlEncode , urlDenode

  • http://meyerweb.com/eric/tools/dencoder/

Online base64_encode, base64_decode

  • http://www.base64decode.org/

这个网站有各种的 Encode, Decode 工具

  • http://www.yellowpipe.com/yis/tools/encrypter/index.php

sha decode

  • http://www.stringfunction.com/sha1-decrypter.html

Unicode to ASCII

  • http://www.ifreesite.com/unicode-ascii-ansi.htm

ROT47

  • http://rot47.net/

Decompiler tool

Flash Decompiler

  • http://www.flash-decompiler.com/

一个 Flash 反组译工具，起级好用，不过这并非免费软体，你可以下载试用版回来使用。

Java Decompiler

  • http://java.decompiler.free.fr/?q=

这是一个小型的 Java 反组译工具，我自已有写一小段 Java 来测试，结果这个工具真的把我写的程式码，一模一样的破解出来，功能真的是非常的强大。

Decompile exe to assembly code

  • http://www.ollydbg.de/

ExeToC Decompiler

  • http://sourceforge.net/projects/exetoc/?source=dlp

Flash exe to swf

  • http://www.northcode.com/blog.php/2007/08/02/Extract-SWF-files-from-Flash-Executables

.Net Decompiler

  • http://www.red-gate.com/products/dotnet-development/reflector/
